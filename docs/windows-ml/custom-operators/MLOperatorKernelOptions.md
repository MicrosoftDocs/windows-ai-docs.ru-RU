# <a name="mloperatorkerneloptions-enum"></a>Перечисление Млоператоркернелоптионс

Указывает параметры, используемые при регистрации ядер пользовательских операторов.

## <a name="fields"></a>Поля

| Имя | Значение | Описание |
|------|-------|-------------|
| Нет | 0 | |
| алловдинамиЦинпутшапес | 1 | Указывает, могут ли фигуры входных десятков зависеть друг от друга в вызовах экземпляра ядра оператора. Если этот параметр не задан, экземпляры ядра могут запрашивать входные тензорные фигуры во время создания, а также работать с инициализацией переднего плана, которые зависят от этих фигур. Установка этого параметра может повысить производительность, если фигуры динамически меняются между операциями вывода, а реализация ядра работает эффективно. |

## <a name="requirements"></a>Требования

| | |
|-|-|

| **Минимальный поддерживаемый сервер** | Windows Server 2019 с возможностями рабочего стола |
