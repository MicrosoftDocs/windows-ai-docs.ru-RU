# <a name="mloperatorexecutiontype-enum"></a>Перечисление MLOperatorExecutionType

Указывает, использует ли ядро ЦП или GPU для вычислений.

## <a name="fields"></a>Поля

| Имя | Значение | Описание |
|------|-------|-------------|
| Не определено | 0 | |
| ЦП | 1 | |
| D3D12 | 2 | |

| | |
|-|-|

| **Header** | MLOperatorAuthor.h |
